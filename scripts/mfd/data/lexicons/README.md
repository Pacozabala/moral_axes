# Moral Foundations Lexicons

This folder contains Moral Foundations Theory (MFT) lexicons used to reproduce the experiments in this thesis.

## Contents

- `MFD_original.csv`  
  Processed version of the original Moral Foundations Dictionary (Graham et al.), as provided by Mokhberian et al.

- `mfd2.0.dic`  
  Moral Foundations Dictionary 2.0, obtained from the Open Science Framework.

- `eMFD_wordlist.csv`  
  Extended Moral Foundations Dictionary (eMFD), a probabilistic lexicon with polarity scores.
